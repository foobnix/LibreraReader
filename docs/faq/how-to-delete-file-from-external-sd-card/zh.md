# 如何从外部SD卡删除书籍

>出于安全原因，系统不允许用户从Android 7+上的外部SD卡删除文件。

在**Librera**中，您需要将SD卡添加为资源(_bookshelf_)，然后允许删除。

* 在“文件夹”标签中，点击文件管理器图标以打开下拉资源列表
* 点击* * +* * 添加书架
* 选择您的SD卡
* 按_SELECT_键以允许从所选位置删除
