# 使用RTL语言阅读


> **Librera**可以很容易地适应阅读从左到右语言的文本，例如阿拉伯语，希伯来语，波斯语或乌尔都语。您只需要更改其默认LTR设置的读取方向即可。


要将读取方向更改为RTL：

* 点击* * 设置* * 图标以打开* * 首选项* * 窗口
* 打开“高级设置”标签
* 点击“阅读方向”链接，然后选择“ _RTL_
* 要调整RTL阅读的两页布局，请选中_强制右页_框
